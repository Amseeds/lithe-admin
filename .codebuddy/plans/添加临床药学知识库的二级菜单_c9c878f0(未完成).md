## 用户需求

在"临床药学知识库"一级菜单下，参考现有"药品信息"二级菜单的结构，添加5个同级的二级菜单项：

- 检验检查
- 护理信息
- 药物相互作用
- 药物不良反应
- 用药指导

## 功能内容

- 扩展菜单配置，在 `fixedMenu` 的 `clinical-pharmacy` children 中添加5个新菜单项
- 创建5个对应的 Vue 页面组件，结构与药品信息组件一致
- 每个页面组件包含基本布局和标题展示

## 技术栈

- 框架: Vue 3 + TypeScript
- 语法: Composition API with `<script setup>`
- 样式: Tailwind CSS

## 实现方案

参考现有"药品信息"的实现模式：

1. 在 `src/api/user.ts` 的 `fixedMenu` 配置中添加菜单项
2. 在 `src/views/clinical-pharmacy/` 目录下创建对应的组件文件夹和 Vue 文件

## 文件命名规则

| 中文名称 | 英文目录名 | 组件名 |
| --- | --- | --- |
| 检验检查 | lab-examination | LabExamination |
| 护理信息 | nursing-info | NursingInfo |
| 药物相互作用 | drug-interaction | DrugInteraction |
| 药物不良反应 | adverse-reaction | AdverseReaction |
| 用药指导 | medication-guidance | MedicationGuidance |
